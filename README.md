# Hohhot 城市信息网站

一个多页面静态网站项目，展示呼和浩特的人口结构、经济、产业、教育与未来规划信息，并附外部来源链接与可视化图表。

## 本地预览
- 直接用浏览器打开 `index.html` 即可。

## 页面结构
- `index.html` 首页总览
- `population.html` 人口结构
- `economy.html` 经济运行
- `industry.html` 重点产业
- `education.html` 教育发展
- `planning.html` 未来规划
- `sources.html` 数据来源

## 可视化实现
- 图表库：Chart.js（CDN）
- 数据文件：`js/data.js`
- 图表渲染：`js/charts.js`

## 部署
- 可部署到 GitHub Pages。
- 推送到仓库后，在仓库设置中启用 Pages（`Deploy from a branch`，分支选择 `main`，目录选择 `/ (root)`）。
