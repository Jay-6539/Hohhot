  • OpenStreetMap + Overpass API（道路与绿地专题数据查询）： https://overpass-turbo.eu/
  • Overpass API 文档（查询语法与接口口径）： https://wiki.openstreetmap.org/wiki/Zh-hans:Overpass_API
  • 行政边界备选 GeoJSON（公开区划边界）： https://geojson.hxkj.vip/
  • 更新时间与说明：地图页采用“Supabase 业态实时拉取优先，查询失败回退本地示例”的策略，并在图例注明。
  • 坐标说明：因高德底图采用 GCJ-02 坐标，叠加专题图层在前端完成 WGS84→GCJ-02 对齐转换。
  • 字段口径：Supabase `huhhot` 表使用 `x/y`（WGS84）与“大类”字段驱动业态网格评分。
