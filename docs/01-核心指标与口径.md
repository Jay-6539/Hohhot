# 呼和浩特城市信息网站：核心指标与口径说明

## 使用规则
- 本文用于统一网站全部图表和结论的统计口径，避免“同名不同义”。
- 所有指标默认采用“自然年”口径（1月1日至12月31日）；人口类如无特别说明采用“年末常住人口”。
- 涉及增速时，需在图表脚注标注“按可比价格”或“按现价”。

## 一、人口结构板块

### P1 常住人口总量
- 定义：在呼和浩特连续居住半年及以上人口。
- 单位：万人。
- 展示方式：近10年折线图。
- 备注：与户籍人口不可混用。

### P2 年龄结构
- 定义：0-14岁、15-59岁、60岁及以上、65岁及以上占比。
- 单位：%。
- 展示方式：分组柱状图 + 年龄金字塔。
- 备注：优先采用人口普查年数据，非普查年可用抽样调查补充。

### P3 抚养比
- 定义：少儿抚养比=(0-14岁人口/15-64岁人口)×100；老年抚养比=(65岁及以上/15-64岁)×100。
- 单位：%。
- 展示方式：双折线图。

### P4 城镇化率
- 定义：城镇常住人口占常住人口比重。
- 单位：%。
- 展示方式：折线图。
- 备注：需与“城区人口占比”区分。

### P5 人口流动净流入
- 定义：净流入人口=流入人口-流出人口（可按区内/区外分层）。
- 单位：万人。
- 展示方式：堆叠柱状图。
- 备注：若口径不连续，需标注“仅作趋势参考”。

## 二、经济板块

### E1 地区生产总值（GDP）
- 定义：一定时期内地区全部常住单位生产活动最终成果。
- 单位：亿元。
- 展示方式：总量柱状图 + 增速折线图。
- 备注：同图须区分“现价总量”与“可比价增速”。

### E2 人均GDP
- 定义：GDP/常住人口。
- 单位：元/人。
- 展示方式：折线图。

### E3 三次产业结构
- 定义：第一、第二、第三产业增加值占GDP比重。
- 单位：%。
- 展示方式：100%堆叠柱图（年度对比）。

### E4 固定资产投资增速
- 定义：固定资产投资完成额同比增长率。
- 单位：%。
- 展示方式：折线图。
- 备注：可拆分基础设施、制造业、房地产、第三产业投资。

### E5 社会消费品零售总额
- 定义：企业（单位）通过交易直接售给个人、社会集团用于最终消费的实物商品金额。
- 单位：亿元。
- 展示方式：柱状图 + 增速线。

### E6 居民人均可支配收入
- 定义：居民可用于最终消费支出和储蓄的总和。
- 单位：元。
- 展示方式：全体/城镇/农村三线对比。

### E7 外贸进出口总额
- 定义：一定时期海关统计的进出口货物总值。
- 单位：亿元。
- 展示方式：进出口双柱图 + 顺差折线。

## 三、产业板块

### I1 重点产业增加值
- 定义：乳业、光伏硅材料、生物医药等重点产业增加值或产值规模。
- 单位：亿元。
- 展示方式：多系列柱状图。
- 备注：若公开口径以“主营业务收入”替代，须在脚注说明。

### I2 重点产业企业数量
- 定义：重点产业中规上企业或高新技术企业数量。
- 单位：家。
- 展示方式：年度柱图。

### I3 产业链完整度（定性指数）
- 定义：按“上游原料-中游制造-下游应用”环节完备程度打分。
- 单位：0-100分（方法自定义）。
- 展示方式：雷达图或评分卡。
- 备注：该指标为研究指标，必须披露评分规则。

### I4 产业就业吸纳能力
- 定义：重点产业就业人数占全社会就业比重。
- 单位：%。
- 展示方式：面积图。

### I5 产业投资强度
- 定义：重点产业投资额/全社会固定资产投资额。
- 单位：%。
- 展示方式：折线图。

## 四、教育板块

### EDU1 在校生规模（分学段）
- 定义：学前、义务教育、高中、中职、高等教育在校人数。
- 单位：万人/人。
- 展示方式：分学段堆叠柱图。

### EDU2 专任教师规模与生师比
- 定义：各学段专任教师数，以及学生数/专任教师数。
- 单位：人、比值。
- 展示方式：双轴图。

### EDU3 教育财政支出
- 定义：财政一般公共预算中的教育支出。
- 单位：亿元。
- 展示方式：柱状图 + 占财政支出比重折线。

### EDU4 高等教育资源
- 定义：高校数量、在校大学生规模、研究生规模（如可得）。
- 单位：所、人。
- 展示方式：指标卡 + 折线图。

### EDU5 人才与产业匹配度（研究指标）
- 定义：高校毕业生专业结构与重点产业岗位需求匹配指数。
- 单位：0-100分。
- 展示方式：桑基图 + 评分卡。
- 备注：需披露数据来源（招聘平台/人社报告）和算法。

## 五、未来规划板块

### F1 规划目标达成率
- 定义：某规划指标实际值/目标值。
- 单位：%。
- 展示方式：仪表盘或子弹图。

### F2 关键工程进度
- 定义：重大项目“开工-在建-投产”节点完成比例。
- 单位：%。
- 展示方式：甘特图或里程碑时间轴。

### F3 创新能力指标
- 定义：高新技术企业数、科技型中小企业数、R&D强度（如可得）。
- 单位：家、%。
- 展示方式：折线图。

### F4 城市综合承载指标
- 定义：交通、医疗、教育、生态等公共服务与资源环境承载变化。
- 单位：多指标复合。
- 展示方式：仪表板。

### F5 预测区间指标
- 定义：人口、GDP、产业结构等在2030年前的预测区间（P10-P90）。
- 单位：与原指标一致。
- 展示方式：扇形区间图（Fan Chart）。

## 六、统一脚注模板（站内强制）
- 数据来源：机构全称 + 文件名 + 发布日期。
- 统计口径：常住/户籍、现价/可比价、是否含所辖旗县区。
- 更新时间：YYYY-MM-DD。
- 说明：口径变更、缺失值处理、估算方法（如有）。
