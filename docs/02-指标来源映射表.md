# 呼和浩特城市信息网站：指标-来源映射表（第一批）

## 说明
- 目标：为每个核心指标绑定“首选来源 + 备选来源”，保证可追溯与可更新。
- 引用优先级：市级统计/政府官网 > 国家统计系统 > 其他公开转载（仅备份）。
- 建议：上线时把本表转换为站内 `sources.json`，供页面脚注自动调用。
