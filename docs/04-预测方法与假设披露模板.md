# 呼和浩特城市信息网站：预测方法与假设披露模板

## 1. 预测模块目标
- 为人口、经济、产业、教育四类核心指标提供“区间预测”，不输出绝对单点承诺。
- 所有预测结果必须附带：模型方法、参数来源、关键假设、不确定性说明。

## 2. 双情景框架定义

### 2.1 基线情景（Baseline）
- 含义：不引入新增重大政策冲击，按历史趋势平滑外推。
- 适用：常住人口、GDP、人均收入、城镇化率、产业占比等连续指标。
- 推荐窗口：近5-10年历史数据（优先连续口径）。

### 2.2 政策情景（Policy）
- 含义：在基线基础上叠加官方规划目标与重大项目投产节奏。
- 输入：规划目标值、项目进度、产业政策强度（可定级）。
- 输出：相较基线的“增量区间”。

## 3. 指标级预测方法

## 3.1 人口类（常住人口、城镇化率、年龄结构）
- 基线方法：
  - 常住人口：`P_t = P_(t-1) * (1 + g_hist)`，`g_hist`取历史平均增速或加权近年增速。
  - 城镇化率：线性趋势 + 上限约束（0~100%）。
- 政策方法：
  - 加入人才引进、产业吸纳就业、住房供给等政策因子调整净流入率。
  - `g_policy = g_hist + delta_policy`。

## 3.2 经济类（GDP、三产结构、收入）
- 基线方法：
  - GDP：复合增长率法（CAGR）或对数线性回归。
  - 人均指标：由总量与人口联动推算。
- 政策方法：
  - 按产业投资与项目投产节奏，调整第二、三产业增速权重。
  - 输出总量区间与结构区间（例如第三产业占比范围）。

## 3.3 产业类（重点产业规模、企业数）
- 基线方法：
  - 按历史产值/营收序列进行趋势外推（可用3年移动平均平滑）。
- 政策方法：
  - 对战略性新兴产业加入政策弹性系数 `epsilon_policy`。
  - `Scale_policy = Scale_baseline * (1 + epsilon_policy)`。

## 3.4 教育类（在校生、教师、支出）
- 基线方法：
  - 在校生：受人口与升学率联动，可用比例法预测。
  - 教育支出：与财政收入或一般公共预算增速联动。
- 政策方法：
  - 纳入新增学校、扩班、教师引进政策的容量变化。

## 4. 区间预测输出规范
- 置信区间：默认 P10/P50/P90 三档。
- 显示方式：扇形区间图（Fan Chart）或阴影带折线图。
- 文案规范：
  - `在基线情景下，2030年{指标}预计为{P10}-{P90}，中位值{P50}。`
  - `在政策情景下，预测区间上移{数值范围}。`

## 5. 参数与数据治理要求
- 所有模型输入需可追溯至“指标-来源映射表”。
- 同一指标跨年份口径变化时，必须进行口径桥接或分段预测。
- 缺失值处理：
  - 优先使用官方补充口径；
  - 其次采用线性插值，并在脚注标注“估算值”。

## 6. 风险与不确定性披露（强制展示）
- 宏观波动风险：外部需求变化导致投资/出口偏离。
- 政策执行偏差：规划目标可能调整或节奏推迟。
- 数据口径风险：统计口径更新导致同比不可比。
- 模型局限：线性或趋势外推不能覆盖突发事件冲击。

## 7. 页面“方法说明卡片”模板

### 模板字段
- 预测对象：`{指标名}`
- 历史样本期：`{YYYY-YYYY}`
- 基线方法：`{CAGR/线性回归/移动平均}`
- 政策调整项：`{政策变量1、2、3}`
- 区间结果：`{P10-P90}`
- 局限说明：`{一句话}`

### 模板示例
- 预测对象：常住人口
- 历史样本期：2015-2024
- 基线方法：加权历史增速外推
- 政策调整项：就业吸纳增强、人才政策强化
- 区间结果：2030年预计 380-402 万
- 局限说明：对跨区域人口流动政策较敏感

## 8. 建议实现（后续开发可直接套用）
- 先用 `CSV + 脚本` 生成预测结果表，再由前端读取展示。
- 每季度更新一次输入数据，自动重算区间。
- 在站内保留“版本号 + 更新时间 + 变更说明”。
