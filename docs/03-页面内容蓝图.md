# 呼和浩特城市信息网站：页面内容蓝图

## 总体导航结构
- 首页（总览）
- 人口结构
- 经济运行
- 重点产业
- 教育发展
- 未来规划
- 数据来源（建议作为公共页，导航常驻）

## 页面统一规范
- 每页结构：`页面摘要 -> 关键结论卡 -> 图表区 -> 口径说明 -> 外部链接`
- 每个图表必须配脚注：来源、口径、时间范围、更新时间。
- 每页至少提供 3 条可点击外部链接（官方优先）。

---

## 1) 首页（总览）

### 页面目标
- 让用户在 60 秒内理解呼和浩特的“人口-经济-产业-教育-规划”全貌。

### 模块与图表
- 模块A：城市概况卡（常住人口、GDP、人均GDP、城镇化率、居民收入）。
- 模块B：趋势总览（近10年人口与GDP双轴图）。
- 模块C：结构总览（三次产业占比 + 年龄结构占比）。
- 模块D：结论卡（现状判断3条 + 趋势判断2条）。

### 结论模板（可复用）
- 现状：`截至{年份}，呼和浩特{指标A}达到{数值}，较上年{上升/下降}{幅度}。`
- 结构：`当前经济结构以{第三产业/第二产业}为主，占比{数值}。`
- 趋势：`在基线情景下，预计到{年份}，{指标B}将处于{区间}。`

### 首页外部链接建议
- 市统计服务：<http://tjj.huhhot.gov.cn/mobile/tjfw/>
- 市发改委经济运行：<http://fgw.huhhot.gov.cn/ywgz/jjyx/>
- 国家统计局普查公报：<https://www.stats.gov.cn/sj/tjgb/rkpcgb/>

## 2) 人口结构页

### 模块与图表
- 模块A：人口总量与增长（折线图，2015-最新年）。
- 模块B：年龄金字塔（最近普查年）。
- 模块C：抚养比变化（少儿/老年双折线）。
- 模块D：民族构成与城镇化率（饼图+折线）。
- 模块E：人口流入流出（区内/区外净流入柱图）。

### 关键结论模板
- `人口总量在{年份区间}保持{增长/波动}，年均增速约{数值}。`
- `人口老龄化程度为{数值}，较基期变化{数值}个百分点。`
- `净流入主要来自{区内/区外}，显示首府吸引力集中在{原因描述}。`

### 必挂外链（示例）
- 国家统计局人口普查：<https://www.stats.gov.cn/sj/tjgb/rkpcgb/>
- 市统计局新闻/专题入口：<http://tjj.huhhot.gov.cn/xwdt/ttxw/index_6.html>

## 3) 经济运行页

### 模块与图表
- 模块A：GDP总量与增速（柱线组合图）。
- 模块B：三次产业结构演进（100%堆叠图）。
- 模块C：投资与消费（固定资产投资增速、社零总额双图）。
- 模块D：收入与就业（城镇/农村收入对比、新增就业）。
- 模块E：外贸表现（进出口总额与结构）。

### 关键结论模板
- `经济总量在{年份}达到{数值}亿元，增速{数值}%，高于/低于{对比对象}。`
- `第三产业对增长贡献率为{数值}%，继续发挥主引擎作用。`
- `投资扩张与消费恢复共同支撑当期增长，但{风险点}需关注。`

### 必挂外链（示例）
- 市统计服务：<http://tjj.huhhot.gov.cn/mobile/tjfw/>
- 市发改委经济运行：<http://fgw.huhhot.gov.cn/ywgz/jjyx/>

## 4) 重点产业页

### 模块与图表
- 模块A：产业地图（乳业、光伏硅材料、生物医药）。
- 模块B：重点产业规模变化（多系列柱图）。
- 模块C：企业数量与创新主体（规上/高新企业趋势）。
- 模块D：产业链完整度评分（雷达图）。
- 模块E：产业风险雷达（市场波动、技术迭代、要素成本）。

### 关键结论模板
- `{产业名}在{年份}形成{规模描述}，是首府产业竞争力核心来源之一。`
- `从产业链环节看，{环节}相对薄弱，建议优先补齐{方向}。`
- `若政策持续加码，预计{产业名}到{年份}规模可达{区间}。`

### 必挂外链（示例）
- 市发改委：<http://fgw.huhhot.gov.cn/>
- 数字呼和浩特“十四五”：<http://hlgena.huhhot.gov.cn/zwgk/zfxxgkzl/fdzdgknr/xqwj/202112/t20211207_1075146.html>

## 5) 教育发展页

### 模块与图表
- 模块A：教育资源总览（学校数、在校生、专任教师指标卡）。
- 模块B：分学段在校生趋势（堆叠柱图）。
- 模块C：教师队伍与生师比（双轴图）。

## 6) 未来规划页

### 模块与图表
- 模块A：规划目标看板（十四五收官进展、十五五方向）。
- 模块B：目标达成率（子弹图）。
- 模块C：重点工程进度（里程碑时间轴）。
- 模块D：双情景预测（基线 vs 政策）。
- 模块E：不确定性说明（政策、外部环境、口径变更）。

### 关键结论模板
- `按现有进度，{目标项}达成率约{数值}%，整体处于{超预期/按计划/滞后}状态。`
- `在政策情景下，到{年份}，{指标}可能达到{区间}，较基线提高{数值}。`
- `预测结果对{关键变量}敏感，需按季度滚动更新。`

## 7) 数据来源页（公共页面）

### 页面目标
- 让读者一键回溯每条核心结论的原始出处。

### 模块设计
- 来源检索框（按指标ID、主题、年份筛选）。
- 来源列表表格（机构、文件、发布日期、链接、适用指标）。
- 数据字典下载（CSV/JSON）。
- 更新日志（每次更新时间与变更说明）。

### 表格字段建议
- `source_id`、`机构`、`文件名`、`发布日期`、`链接`、`覆盖指标ID`、`口径说明`、`最后验证时间`。
